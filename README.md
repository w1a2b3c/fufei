# 项目说明

## 环境要求
- PHP 版本：**7.2**
- 扩展要求：**sg11** 扩展

## 配置说明
1. **伪静态**：使用 **ThinkPHP** 的伪静态规则。
2. **数据库配置**：在 `config/database.php` 文件中修改数据库连接信息。
3. **安全设置**：关闭防跨站（XSS 防护）。
